# Run q1 using cli
```
export PATH_TO_FX=lib
javac --module-path $PATH_TO_FX --add-modules javafx.controls,javafx.fxml q1/*/*.java q1/Main.java
java --module-path $PATH_TO_FX --add-modules javafx.controls,javafx.fxml q1.Main
```
